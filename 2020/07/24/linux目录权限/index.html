linux调整文件及目录权限

No tag
Publish Date:   2020-07-24

useradd tom == 新建一个组，名为tom
id tom == 查看tom的uid值
su tom == 切换到tom用户模式下
echo “xxx” > /tmp/test.txt == 把xxx写入test.txt下

1、# ls -l test.txt == 查看文件的权限信息
-rw-r–r–. 1 root(所属者） root(所属组） test.txt
1 = 节点

2、# ls -ld testdir/ == 查看目录的权限信息
drwxr-xr-x. 2 root root testdir/

  • rw- r– r– . == 第一个-为第一字段，后面每三个为一字段
    d rwx r-x r-x . == 第一个-为第一字段，后面每三个为一字段
    字段1：文件类型
    -普通文件 
 d目录
 l符号链接
 b块设备

字段2：文件所属者对该文件的权限
r w x
文件：read=读取文件，write=写入文件， 可执行权限（不加x为不可执行文件）
目录：r=查看目录内容，w=可以增删文件 ，可以进入目录

字段3：文件所属组的权限，如root组，tom组

字段4：其他用户的权限

chmod（change mode）对象 算数运算符 权限 文件
例：chmod o-r /tmp/test.txt == 删除o用户的r权限
对象：u（所属者）， g（所属组）， o（其他用户的权限）， a （all）
算数运算符：-（清除权限）， +（附加权限）， =（指定权限）
权限：r w x

chmod u+w 文件

0 000 —
1 001 –x
2 010 -w-
3 011 -wx
4 100 r–
5 101 r-x
6 110 rw-
7 111 rwx

4r 2w 1x
rwx(4+2+1) r– —
chmode 740 文件

改变文件的所属者为tom，所属组改为tom组
chown tom /tmp/test.txt == 改变所属者
chgrp tom /tmp/test.txt == 改变所属组

ll -d 文件 == 查看文件权限
rm -fr 文件 == 删除文件
which vim == 查找vim的目录
find /usr/bin -perm 2755 == 查找usr/bin目录下权限为2(1-粘滞位，2-stid，4-suid）755的文件

粘滞位 sgid suid 权限

粘滞位针对目录赋权，目录中创建的文件只有建立者可以删除
chomd o+t 目录 == 赋予其他用户t权限
ll -d 目录 == 查看目录最后一权限为t

sgid：针对目录建立的权限，在该目录中建立的文件所属组继承父目录的所属组
chomd g+s 目录 == 赋予所属组t权限
ll -d 目录 == 查看目权限为sgid

suid：针对可执行文件建立
chmode u+s 文件 == 赋予所属者suid权限

不允许添加新用户的请求
/etc/group
/etc/passwd
/etc/shadow
/home/xxxx

chattr +i /etc/group /etc/passwd /etc/shadow == 不允许添加新用户的请求
chattr -i /etc/group /etc/passwd /etc/shadow == 允许添加新用户的请求
